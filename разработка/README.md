## Быстрые решения

- **Платформа:** Telegram Mini App
- **Аудио (TTS/STT):** не делаем в v1
- **Цвета:** армянский флаг (красный · синий · оранжевый)
- **Язык перевода:** Русский или English — **только в Настройках**
- **Диалект:** Восточный или Лорийский — **только в Настройках**
- **UI v3:** без лишних кнопок — свайпы, одна CTA на папку
