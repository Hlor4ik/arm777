# Дизайн-система — цвета армянского флага

Все цвета интерфейса строятся на трёх полосах флага Армении: **красный · синий · оранжевый**.  
Задача — не «залить всё тремя цветами», а выдержанная палитра: синий как основа, оранжевый как действие, красный как акцент и статус.

---

## 1. Базовые цвета флага

| Название | HEX | RGB | Роль |
|----------|-----|-----|------|
| **Flag Red** | `#D90012` | 217, 0, 18 | Ошибки, важные бейджи, streak |
| **Flag Blue** | `#0033A0` | 0, 51, 160 | Основной бренд, заголовки, активные элементы |
| **Flag Orange** | `#F2A800` | 242, 168, 0 | Кнопки CTA, прогресс, «Открыть доступ» |

Официальные оттенки могут чуть отличаться — для UI используем эти как канон проекта.

---

## 2. Тёмная тема (основная для Telegram)

Telegram Mini App чаще открывается в тёмной теме — фон **не чёрный**, а **глубокий синий** (производный от полосы флага).

| Токен | HEX | Использование |
|-------|-----|---------------|
| `--bg-base` | `#0B1426` | Фон приложения |
| `--bg-elevated` | `#121F38` | Карточки, панели |
| `--bg-surface` | `#1A2D4F` | Поднятые карточки, модалки |
| `--bg-hover` | `#243B66` | Hover / pressed |

| Токен | HEX | Использование |
|-------|-----|---------------|
| `--text-primary` | `#F4F6FA` | Основной текст |
| `--text-secondary` | `#9AABC4` | Подписи, метки |
| `--text-muted` | `#6B7F9E` | Placeholder, disabled |

---

## 4. Семантические цвета

| Статус | Цвет | Пример |
|--------|------|--------|
| Успех / сдано | `#F2A800` + иконка ✓ | Папка «Приветствия» сдана |
| В процессе | `#3D5FB8` | Progress bar |
| Заблокировано | `#4A5568` + opacity 0.5 | Закрытые папки |
| Правильный ответ | `#2E7D52` на `--bg-elevated` | Экзамен (мягкий зелёный, не из флага — только для UX) |
| Неправильный ответ | `#D90012` | Экзамен |

> Зелёный для «правильно» — единственное отступление от флага: иначе «верно» и «ошибка» оба будут красно-оранжевыми и путают. Оттенок приглушённый.

## 5. Градиенты (декоративные)

```css
/* Шапка / hero-блок */
--gradient-flag: linear-gradient(
  180deg,
  #D90012 0%,
  #D90012 33.33%,
  #0033A0 33.33%,
  #0033A0 66.66%,
  #F2A800 66.66%,
  #F2A800 100%
);

/* Мягкий фон карточки «Алфавит» */
--gradient-card-alphabet: linear-gradient(
  135deg,
  #121F38 0%,
  #1A2D4F 50%,
  rgba(0, 51, 160, 0.25) 100%
);

/* Кнопка CTA */
--gradient-cta: linear-gradient(
  180deg,
  #FFBD33 0%,
  #F2A800 50%,
  #C98900 100%
);
```

Полоса флага — **только** в логотипе, splash, onboarding (1–2 экрана). Не на каждой карточке.

## 6. Компоненты

### Нижняя навигация (4 вкладки)
- Фон: `--bg-elevated` + border-top `#243B66`
- Неактивная иконка: `--text-muted`
- Активная: `--accent-orange` иконка + подпись `--accent-primary`

### Карточка папки
- Фон: `--bg-elevated`, border `1px solid #243B66`
- Progress bar: track `#1A2D4F`, fill градиент blue → orange
- Кнопка «Открыть доступ»: `--gradient-cta`, текст `#0B1426` (тёмный на оранжевом)
- Заблокированная: opacity 0.45, lock icon `--text-muted`

### Карточка режима
- Иконка в круге: фон `rgba(0, 51, 160, 0.35)`, icon `#F2A800`
- Алфавит — особая: тонкая полоска `--gradient-flag` сверху (4px)

### Flashcard
- Карточка: `--bg-surface`, shadow `0 8px 32px rgba(0, 0, 0, 0.35)`
- Тап — flip; свайп влево/вправо — «ещё раз» / «знаю»
- **Без** переключателя диалекта и **без** кнопок внизу (см. v3)

### Экзамен
- Таймер: `--accent-red` если < 60 сек
- Варианты ответа: default `--bg-hover`, selected border `#0033A0`, wrong `#D90012`, correct `#F2A800`

## 7. Типографика

| Уровень | Размер | Weight | Цвет |
|---------|--------|--------|------|
| H1 | 24px | 700 | `--text-primary` |
| H2 | 18px | 600 | `--text-primary` |
| Body | 16px | 400 | `--text-primary` |
| Caption | 13px | 400 | `--text-secondary` |
| Transcription | 22px | 500 | `#F2A800` (слова на карточках) |

Шрифт: системный stack Telegram (`-apple-system`, `SF Pro`, `Roboto`) — без кастомных файлов в v1.

## 9. Интеграция с Telegram theme

```css
/* Fallback на Telegram, если доступно */
background: var(--tg-theme-bg-color, var(--bg-base));
color: var(--tg-theme-text-color, var(--text-primary));
```

Приоритет: **наши токены флага** поверх `--tg-theme-*`, чтобы бренд был узнаваемым. Кнопка MainButton бота: `#F2A800`.
