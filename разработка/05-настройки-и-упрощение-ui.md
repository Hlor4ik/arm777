# Настройки и упрощение UI (v3)

> Решение из чата · 04.07.2026

---

## 1. Настройки — единственное место выбора языка и диалекта

Все переключатели **только** на вкладке «Настройки». На экранах учёбы их **нет**.

### Блок «Язык перевода»

Язык, **с которого** учим армянский (помимо транскрипции):

| Вариант | Что показывается на карточках и в заданиях |
|---------|---------------------------------------------|
| **Русский** | RU ↔ транскрипция |
| **English** | EN ↔ транскрипция |

- Интерфейс приложения (меню, кнопки, подписи) **следует** выбранному языку.
- Одновременно RU и EN в заданиях не показываем.

UI: сегмент **Русский | English** (2 pill, без лишних кнопок).

### Блок «Диалект армянского»

| Вариант | Транскрипция |
|---------|--------------|
| **Восточный** | `transcription_eastern` |
| **Лорийский** | `transcription_lori` |

UI: сегмент **Восточный | Лорийский**.

> Режима «оба диалекта» нет — один выбор в настройках, применяется везде.

### Остальное в настройках

- О приложении (версия, ссылка)
- Сброс прогресса (одна outline-кнопка, подтверждение)

**Не делаем:** отдельный переключатель «язык интерфейса» — он совпадает с языком перевода.

## 2. Убранные элементы (лишние кнопки)

| Было | Стало |
|------|-------|
| Pills «Восточный \| Лорийский» на карточках | Убрано → только в Настройках |
| Кнопки «Знаю» + «Ещё раз» под карточкой | **Свайп** вправо = знаю, влево = ещё раз; тап = перевернуть |
| Кнопка «Далее» на экзамене | **Тап по ответу** → подсветка → автопереход (~0,8 с) |
| Экран детали папки с «Начать» / «Экзамен» | **Нет отдельного экрана** — действия на строке папки |
| Кнопка «Слабые места» на Прогрессе | Только режим на вкладке «Режимы» |
| Бейдж «Без аудио» на карточках режимов | Убран (и так не делаем аудио) |
| Дублирующие CTA на одной карточке папки | **Одна** кнопка по статусу (см. ниже) |

---

## 3. Папки — одна кнопка на карточку

| Статус | Элемент на карточке | Действие по тапу |
|--------|---------------------|------------------|
| 🔒 Заблокирована | Lock, без кнопки | Ничего / toast «Сдайте предыдущую папку» |
| 🟡 Доступна | **«Открыть доступ»** (оранж) | Добавляет слова в базу |
| 📖 В изучении | Progress bar, без кнопки | → вкладка «Режимы» (или последний режим) |
| ✅ Сдана | Бейдж ✓ | Повторное изучение через «Режимы» |
| 📝 Экзамен готов (≥70%) | Маленькая иконка 📝 справа | Тап по иконке → экзамен |

Экзамен **не** отдельная большая кнопка — только иконка, когда готовность ≥ 70%.

## 4. Карточки (режим)

```
┌─────────────────────────┐
│  ←  Карточки · Привет   │  12/30
├─────────────────────────┤
│                         │
│      [  Привет  ]       │  ← tap = flip → barev
│                         │
│   ← ещё раз   знаю →    │  ← подсказка свайпа, не кнопки
└─────────────────────────┘
```

- Нет pills диалекта
- Нет кнопок внизу
- Haptic на свайп (Telegram)

## 7. Влияние на данные

```json
{
  "settings": {
    "baseLang": "ru",
    "dialect": "eastern"
  }
}
```

Слово в базе:

```json
{
  "ru": "Привет",
  "en": "Hello",
  "transcription_eastern": "barev",
  "transcription_lori": "barev"
}
```

Режимы и экзамен используют `baseLang` для стороны «перевод», `dialect` — для транскрипции.
