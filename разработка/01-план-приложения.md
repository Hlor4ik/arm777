# Предварительный план — Telegram Mini App «Армянский»

> Сохранено из обсуждения · 04.07.2026

---

## Концепция

Mini App в Telegram для изучения армянского: **~2 500–3 000 слов** в **15 тематических папках**, **4 вкладки**, прогрессия **открыть → учить → экзамен → следующая папка**.

| Параметр | Значение |
|----------|----------|
| Платформа | Telegram Mini App |
| Язык перевода | **Русский или English** (выбор в Настройках) |
| Диалект | Восточно-армянский или лорийский (выбор в Настройках) |
| Письмо | Только режим «Алфавит» |
| Аудио TTS/STT | **Не делаем в v1** |
| Цвета UI | Армянский флаг → см. [02-дизайн-система.md](./02-дизайн-система.md) |

---

## 4 вкладки

| Вкладка | Назначение |
|---------|------------|
| **Режимы** | Способы учить открытые слова |
| **Папки** | 15 тем, «Открыть доступ», блокировки |
| **Прогресс** | Статистика, streak, слабые слова, готовность к экзамену |
| **Настройки** | Язык перевода (RU/EN), диалект, сброс |

---

## User Flow

```
Алфавит (экзамен 85%+) 
  → Папка «Приветствия» [Открыть доступ]
  → Режимы изучения
  → Экзамен папки (80%+)
  → Следующая папка разблокирована
  → …
```
