# Реализация v1 — 04.07.2026

## Статус

Приложение реализовано по плану. Production build проходит успешно.

## Структура кода

- `app/` — React + Vite + TypeScript Mini App
- `data/` + `scripts/` — 2540 слов, алфавит, validate/build
- `vercel.json` — SPA rewrites

## Команды

```bash
npm run dev       # разработка
npm run build     # data + production build
npm run validate  # проверка словаря
```

## Деплoy Vercel

1. Push на GitHub
2. Import в Vercel: Build `npm run build`, Output `app/dist`
3. BotFather → Menu Button → URL проекта
