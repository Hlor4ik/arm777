# Режимы и экраны

> Дополнение к [01-план-приложения.md](./01-план-приложения.md)

## Экран: Папки

**Элементы:**
- Список 15 карточек тем
- Статусы: 🔒 / «Открыть доступ» / в изучении / ✅ сдана
- Progress bar (синий → оранжевый градиент)
- **Одна CTA на карточку** — см. [05-настройки-и-упрощение-ui.md](./05-настройки-и-упрощение-ui.md)
- Экзамен: маленькая иконка 📝 справа, когда готовность ≥ 70%
- Нет экрана «детали папки» с отдельными кнопками

**Цвета:**
- CTA «Открыть доступ» — оранжевый градиент `#F2A800`
- Сдана — бейдж оранжевый с ✓
- Заблокирована — серо-синий, opacity 45%

**Мокап (актуальный v3):** [фото/v3-folders-screen-mockup.png](./фото/v3-folders-screen-mockup.png)

## Экран: Режимы

**Сетка режимов** (2 колонки). Каждая карточка:
- Иконка в синем круге, accent оранжевый
- Название + короткое описание
- Режим «Алфавит» — полоска флага 4px сверху

**Недоступные режимы** (папка не открыта): disabled + подсказка.

**Мокап (актуальный v2):** [фото/v2-modes-screen-mockup.png](./фото/v2-modes-screen-mockup.png)

## Экран: Карточки

- Центральная flashcard на `--bg-surface`
- Лицо: RU или EN (из настроек) / оборот: транскрипция (`--text-transcription` оранжевый)
- **Тап** — перевернуть; **свайп** вправо = знаю, влево = ещё раз
- **Без** pills диалекта и **без** кнопок внизу
- Прогресс: «12/30», bar blue→orange
- Диалект и язык перевода — только в [Настройках](./05-настройки-и-упрощение-ui.md)

**Мокап (актуальный v3):** [фото/v3-flashcard-mode-mockup.png](./фото/v3-flashcard-mode-mockup.png)

---

## Экран: Экзамен

- Header: название папки, таймер, progress
- Вопрос крупно (RU или EN из настроек)
- 4 варианта (кнопки `--bg-hover`)
- **Тап по ответу** → подсветка → автопереход (~0,8 с)
- **Без** кнопки «Далее»

**Мокап (актуальный v3):** [фото/v3-exam-screen-mockup.png](./фото/v3-exam-screen-mockup.png)

---

## Экран: Прогресс

- Streak (огонь + число, accent red/orange)
- Круговая диаграмма: выучено / в работе / слабые
- Список папок с % готовности к экзамену
- **Без** кнопки «Слабые места» — режим только на вкладке «Режимы»

## Экран: Настройки

Единственное место выбора языка и диалекта — см. [05-настройки-и-упрощение-ui.md](./05-настройки-и-упрощение-ui.md):

- **Язык перевода:** Русский | English (интерфейс + сторона карточки)
- **Диалект армянского:** Восточный | Лорийский
- О приложении
- Сброс прогресса (outline red)

## Режим «Алфавит» — подрежимы

| Подрежим | Задача |
|----------|--------|
| Буквы | Карточка: Ա → ayb → «a» |
| Звук → буква | Транскрипция → выбор буквы |
| Буква → звук | Буква → выбор транскрипции |
| Слоги | ma, ba, ga… |
| Мини-слова | 2–3 буквы |

## Навигация

```
Bottom tabs (всегда видны)
├── Режимы → [выбор режима] → [сессия] → результаты
├── Папки → [детали папки] → Открыть / Экзамен
├── Прогресс → детальная статистика
└── Настройки
```

BackButton Telegram — на вложенных экранах.
